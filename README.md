## 🕹️ Tetris
  This Tetris was created with JavaScript, in 9 days. Thanks to this project I learned JavaScript quickly.

## 🕹️ How to Play
  download the repository, open index.html and play!

  W -> Rotate piece
  
  A -> Move to Left
  
  D -> Move to Right
  
  S -> More Speed
